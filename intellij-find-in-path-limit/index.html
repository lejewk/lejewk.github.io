intellij find in path 검색 제한 설정

Posted on 2019-04-22 Edited on 2020-03-06

개요

Find in path 도구의 검색 제한 설정을 해제하여 많은 양의 검색결과를 볼 수 있습니다.
기본 검색 사이즈는 파일 수와 상관없이 100 라인에 대해서만 검색하므로,
동일 파일에 원하는 검색단어가 많이 있을수록 다른파일을 찾지못해 효율이 떨어집니다.

설정

shift, shift (두번 입력) 후 Registry 를 검색합니다.

ide.usages.page.size 를 찾아 값을 원하는만큼 변경합니다.
