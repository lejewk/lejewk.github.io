go module(mod) 를 이용한 패키지 관리 방법

Posted on 2020-01-12 Edited on 2020-03-06

요약

go mod init 을 이용해 프로젝트 루트 경로에 go.mod 파일 생성하면 됩니다.

mod 파일 생성

go.mod 파일은 루트 경로에 하나만 존재하면 됩니다.
프로젝트 내 디렉토리별로 go.mod 를 만들 필요가 전혀 없습니다.
